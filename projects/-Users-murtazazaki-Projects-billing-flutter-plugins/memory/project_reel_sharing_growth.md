Reel sharing is a key growth flywheel for the app, deferred from the Tutorial Engine v5 placeholder phase.

**Why:** Sharing drives organic acquisition. Similar to shared ledger pages served via microservices, video sharing should use the shorts microservice on Cloudflare Pages — simple, easy to implement, CDN-backed.

**How to apply:** When implementing share button on reels (post-launch), use Cloudflare Pages to host shareable reel landing pages at `shorts.gstapps.com`. Pattern mirrors shared ledger page architecture. Track with `tutorial_video_shared` analytics event.
